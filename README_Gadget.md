# 🛒 Gadget

A full-stack e-commerce web application for buying and selling electronic gadgets — built with a clean shopping experience similar to modern e-commerce platforms.

🔗 **GitHub:** [github.com/vyshnavii-k/ECommerceApp](https://github.com/vyshnavii-k/ECommerceApp)

---

## ✨ Features

### 🔐 User Authentication
- Secure login and logout
- Personalized session for each user

### 🛍️ Shopping Experience
- Browse electronic gadget listings
- Product details and descriptions
- Add to cart and manage orders
- Clean, intuitive storefront UI

## 🚀 Getting Started

### Prerequisites
- Node.js v16+

### Installation

```bash
# Clone the repository
git clone https://github.com/vyshnavii-k/ECommerceApp.git
cd ECommerceApp

# Install dependencies
npm install

# Start the server
node server.js
```

Then open `http://localhost:3000` in your browser.
